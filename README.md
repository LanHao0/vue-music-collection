# musicr
vue写的音乐收藏库，在线预览：https://www.lanhaoo.club/musicr/#/all

1. 点击专辑可以进入该专辑
2. 专辑页面点击专辑图可以返回
3. 支持A-Z排序，年份排序，默认年份排序


因为自己换了音乐软件，所以旧版发出来了。  
此版本包含我2020年前听的歌曲。
